# SVIROXK — статус компилятора

## Этап 1 — завершён

Базовый язык и LLVM-компилятор готовы к использованию. Этап 2 (самохостинг) — следующий, когда решишь.

## Сделано

| Область | Статус |
|---------|--------|
| Лексер + номера строк в ошибках | ✅ |
| Парсер, приоритет `*`/`/` выше `+`/`-` (символы `+ - * /` не менялись) | ✅ |
| Переменные, массивы, условия, циклы, функции | ✅ |
| Указатели, файлы, импорт, ввод/вывод | ✅ |
| Структуры `tip` + экземпляр `svi tip Player lor p` + `p.field` | ✅ |
| Типы параметров `rox`/`dor`/`cos`/`ptr` в `fucn` | ✅ |
| `rep()` — базовая замена подстроки | ✅ |
